### 1. 寻址接口

#### GET http://ip:11096/inner/clientAddressing/interfaceContract?s.weather-base.gy

**请求参数**
- s.weather-base.gy：服务标识（string，必填）

**响应示例**
```json
{
    "datas": [
        {
            "ip": "192.168.1.200",      // 服务IP
            "portInfoList": [
                { "port": 11096 },       // 服务端口
                { "port": 12345 }        // 备用端口
            ]
        },
        {
            "ip": "127.0.0.1",         // 备用IP
            "portInfoList": [
                { "port": 8888 },        // 备用端口
                { "port": 9999 }         // 备用端口
            ]
        }
    ],
    "respCode": 0,                       // 响应码，0表示成功
    "respMsg": "寻址成功."                // 响应信息
}
```

### 5. 获取水平流

#### POST http://ip:port/weather/sea/horizontalStream

**请求体（application/json）**
```json
{
    "startLon": 120.1,      // 起始经度
    "endLon": 121.5,        // 结束经度
    "startLat": 23.5,       // 起始纬度
    "endLat": 25.0,         // 结束纬度
    "level": 10,            // 深度层级，0代表不过滤
    "startTime": "2025-03-31 00:00:00", // 开始时间
    "endTime": "2025-03-31 23:59:59"    // 结束时间
}
```

**响应体（application/json）**
```json

    "endLon": 120,

            "dimName": "time",
```
